# Portal Notícia — Site de Notícias

Bem‑vindo ao repositório do site **Portal Notícia** — um portal de notícias online com artigos, seções diversas e publicação de conteúdos jornalísticos.

##  Visão Geral

Este site apresenta:

- Página inicial com os artigos mais recentes e destaques.  
- Páginas para diferentes categorias de notícias (ex: política, saúde, tecnologia, etc.).  
- Layout responsivo com HTML, CSS (e possivelmente JavaScript).  
- Estrutura de site estático, pronto para ser hospedado via GitHub Pages — como feito atualmente. :contentReference[oaicite:2]{index=2}  

O site está publicado em:  
https://linosimango27-crypto.github.io/portalNoticia/

##  Tecnologias / Ferramentas Utilizadas

- HTML + CSS (e possivelmente JavaScript).  
- Git + GitHub para controle de versão e hospedagem via GitHub Pages. :contentReference[oaicite:3]{index=3}  
- Arquivos estáticos — não há backend: todo o site roda no cliente.
